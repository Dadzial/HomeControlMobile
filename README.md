# HomeControl Mobile

A modern mobile application for smart home control and automation, built with Expo.

## Features

- **Remote Control**: Control your smart home devices from anywhere
- **Real-time Monitoring**: Get live updates from your smart home sensors
- **Automation**: Create custom automation rules for your smart devices
- **Multi-platform**: Available for both iOS and Android
- **Secure**: End-to-end encryption for all communications
- **Expo**: Easy development and testing with Expo Go

## Prerequisites

- Node.js 16.0.0 or later
- npm 8.0.0 or later
- Expo CLI: `npm install -g expo-cli`
- Expo Go app (for testing on physical devices)
- Android Studio (for Android development)
- Xcode (for iOS development, macOS only)
